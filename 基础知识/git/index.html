git ​

安装配置 ​

git for windows 安装后通过以下命令查看是否安装成功：

javascript
git --version

简单命令 ​

  • pwd ：查看当前目录
  • ls ：查看当前目录下的文件
  • cd : 切换目录
  • ll ：查看当前目录下的文件
  • dir ：查看当前目录下的文件
  • mkdir ：创建文件夹
  • touch ：创建文件
  • ls -a :查看隐藏的文件（默认.开头为默认隐藏文件）
  • rm -rf :删除文件 [谨慎使用]

  • git config --global alias.co checkout

branch 分支 ​

  • git branch 查看分支
  • git branch xxx 创建分支
  • git checkout xxx 切换分支
  • git merge xxx 合并分支

  • git branch -D xxx 强制删除分支

问题场景 ​

在一个分支中有修改，未提交的文件，这时候我们想到另一个分支去处理修改bug？ ​

git stash
//接着切换到 另一个分支进行处理问题
//最后切换回来
git stash apply
//这是可以继续使用
